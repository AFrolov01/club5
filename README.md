# ⚔️ ClanWar Bot

Telegram-бот войны кланов с дуэлями на минном поле (5×5).

## Структура

```
clanwar_bot/
├── main.py           — точка входа, запуск бота
├── database.py       — работа с SQLite (aiosqlite)
├── game_logic.py     — множители, генерация поля, клавиатуры
├── matchmaking.py    — система подбора соперников
├── scheduler.py      — автоматические дуэли (каждые 2 дня) + AFK
├── handlers_clan.py  — /createclan, /join, /clan, /top
├── handlers_duel.py  — /minduel, выбор мин, нажатие клеток, сбор очков
├── data/             — папка для clanwar.db (создаётся автоматически)
└── requirements.txt
```

## Установка

```bash
pip install -r requirements.txt
python main.py
```

## Команды

| Команда | Описание |
|---|---|
| `/start` | Запустить бота в группе |
| `/createclan` | Создать клан (имя → аватар → девиз) |
| `/join` | Вступить в клан (листать карточки ⬅️ ➡️) |
| `/clan` | Инфо о своём клане |
| `/top` | Рейтинг всех кланов |
| `/minduel` | Принять участие в объявленной дуэли |
| `/forceduel` | [Только admin] Принудительно вызвать дуэль |

## Как работает

1. Бот добавлен в группу → регистрирует её
2. Каждые **2 дня** автоматически выбираются 2 игрока из разных кланов и объявляется дуэль
3. Каждый из них жмёт `/minduel` → в личке выбирают количество мин (1–6)
4. Открывают клетки на поле 5×5 → множители растут с каждой безопасной клеткой
5. Нашёл мину → очки клана ×0.75; забрал сам → очки клана × текущий множитель
6. Если игрок АФК >30 минут → автоматический штраф и передача хода
7. Война кланов длится **30 дней**, затем объявляется победитель и всё сбрасывается до 100 Al

## Множители (пример, 6 мин)

`x1.28 ➡️ x1.70 ➡️ x2.30 ➡️ x3.17 ➡️ x4.43 ➡️ ...`

Чем больше мин — тем выше риск, тем выше награда.
